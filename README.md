# 🎬 Netflix SQL Data Analysis

## 📌 Project Overview

This project analyzes the Netflix Titles dataset using PostgreSQL to uncover meaningful business insights. It demonstrates SQL skills ranging from beginner to advanced, including data exploration, data cleaning, aggregations, subqueries, Common Table Expressions (CTEs), and window functions.

The project is designed to simulate real-world business scenarios and showcase SQL techniques commonly used by Data Analysts.

---

## 🎯 Objectives

- Analyze Netflix Movies and TV Shows.
- Explore content trends over the years.
- Identify top-performing countries and directors.
- Compare Movies vs TV Shows.
- Analyze ratings, genres, and release years.
- Solve business problems using SQL.
